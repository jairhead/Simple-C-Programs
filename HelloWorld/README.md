# Hello World
Hello World in C#

## Description
An extremely basic C# program that was written just to get something to compile & run in C#!

## Compile & Run Instructions (Git Bash)
* `cd` into the simple program directory (e.g. HelloWorld)
* `csc <program name>` to compile the program
* `./<program name>` to run the program

## Compile & Run Instructions (Windows-Friendly)
* Using File Explorer, navigate to the `HelloWorld` directory within the repository
* Right-click inside the directory, and select `Open in Terminal`
* Inside the terminal, run `csc HelloWorld.cs` to compile the executable
* Inside the terminal, run `./HelloWorld.exe` to run the executable
